B Good

A Ruby on Rails app to make volunteering in Miami fun and easy! Features include:

  • Auth0 passwordless login using SMS
  • Web API from Volunteermatch.org to find local opportunities.
  • JavaScript powered modal to add sign ups to your dashboard
  • Facebook and Twitter social plugins to share your activity & invite friends
  • Styled using Materialize CSS framework

    • Try it out: B Good Miami
    • Repo: Github

    Space Out

    A Ruby on Rails dashboard app to help you focus. Take a coding break, get inspired with a daily photo, quote, and music. Features include:

  • Secure login powered by a Bootstrap modal, using JavaScript
  • Web API delivers a new, design-based quote each visit. Floating effect using JavaScript.
  • Background is a daily space image from the Hubble Space Telescope. Powered using the NASA space images Web API
  • Spotify widget plays the playlist of your choice
  • Twitter social plugin to share the quote

    • Try it out: Space Out
    • Repo: Github
